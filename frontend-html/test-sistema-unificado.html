⚠️ Instrucciones para Desarrolladores

SIEMPRE usar estas funciones:

  • window.session.isLoggedIn() - Para verificar sesión
  • window.session.getUser() - Para obtener usuario
  • window.session.updateHeader() - Para actualizar header
  • window.session.logout() - Para cerrar sesión

NUNCA usar localStorage.getItem('token') directamente.
