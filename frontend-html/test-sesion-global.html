📋 Estructura del Objeto window.session

Método Implementación Uso
getUser() JSON.parse(localStorage.getItem("user")) Obtener datos del usuario
getToken() localStorage.getItem("token") Obtener token de autenticación
isLoggedIn() !!localStorage.getItem("token") Verificar si hay sesión activa
logout() removeItem + redirect Cerrar sesión y redirigir
updateNavbar() Control de elementos DOM Actualizar menú según rol
setSession() setItem + updateNavbar Establecer sesión
